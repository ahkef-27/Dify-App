# Dify-App

## Dify-App: Auto Research & Analysis System
## Web情報収集・LLM分析・可視化を統合した自律型インテリジェンス・プラットフォーム

Google Apps Script (GAS) と Dify (LLM Workflow) を高度に連携させ、特定トピックに関する最新情報の収集から、多角的なAI分析、そしてレポーティングまでを完全無人化した資産運用型リサーチシステムです。

ユーザーがキーワードを一度設定するだけで、

### ・深夜帯の自動Webリサーチ（StartIndexページング対応）

### ・URL重複排除とクレンジングによるコスト最適化

### ・Dify AIによるセンチメント・トレンド・競合分析

### ・Slack通知とWebダッシュボードによるシームレスな可視化

これらがすべて自律的に実行されます。
「検索」という労働から人間を解放し、「意思決定」にリソースを集中させたいビジネスパーソンやリサーチャーに最適です。

## 1. 検索深化ロジック（Deep Search Protocol）

Google Custom Search APIの仕様を超え、startIndexによる自動ページめくり機能を実装。目標とするデータ件数に達するまでリサーチを続行し、常にボリュームのある分析母数を確保します。

## 2. Client-First なWebダッシュボード（Visual Intelligence）

index.html により、以下の機能を備えた専用UIを提供します。

    セキュアなログインゲート: ブランド感のあるグラデーションUIを採用した認証画面。

    感情分析の可視化: Google Charts を用いたドーナツチャートにより、市場のポジネガを直感的に把握。

    ハッシュタグ・タグクラウド: AIが抽出した trend_words を動的に生成し、キーワードの熱量を視覚化。

## 3. インテリジェント・重複排除

URL列をピンポイントで照合し、trim()（空白削除）を徹底。同一記事を二度AIに投げない、コスト意識の高い設計です。

## 4. 「JST Timezone Lock」による信頼性

全データを日本標準時（JST）で固定出力。ビジネス報告資料としての時間の整合性を完全に維持します。

### 🎨 デザイン・UIのこだわりポイント

### 1. 心理的な安心感を与える配色

    グラデーション (linear-gradient): ログイン画面に紫〜青のグラデーションを使うことで、モダンで高級感のある「SaaS（サービス）」のような第一印象を与えています。

    成功の色 (#28a745): 随所に緑色を配し、AIリサーチが正常に完了している「ポジティブな状態」を視覚的に刷り込んでいます。

### 2. 「カプセル型」セパレーターの採用

CSS

capsule-line { border-radius: 10px; opacity: 0.2; }

通常の hr（直線）ではなく、端を丸めた太めのラインに透過度（opacity）を持たせることで、**「堅苦しすぎない、洗練されたビジネスツール」**の雰囲気を演出しています。

### 3. 動的なインタラクション

Google Charts: 感情分析を単なる数字ではなく、アニメーション付きのドーナツチャートで表示することで、データの「納得感」を高めています。

ハッシュタグ表示: badge rounded-pill を使ったタグ形式の表示は、SNSに慣れた現代のユーザーにとって直感的に「今、何が話題か」を理解させる優れたUIです。

### ⚙️ システム的な工夫

    UXの向上: エンターキーでのログイン操作や、読み込み中の「データを読み込んでいます...」というプレースホルダーなど、ユーザーを待たせない工夫が凝らされています。

    スケーラビリティ: max-width: 1200px の設定により、PCの大画面で見ても間延びせず、タブレットで見ても崩れない絶妙なサイズ感が保たれています。
